Cura

Cura is a systems administration application for the Android platform

Download as .zip Download as .tar.gz View on GitHub

Cura is a mobile phone application for the Android platform (Gingerbread 2.3.3 and above). It is a bundle of systems administration tools targeted mainly at sysadmins.

When in the hands of a sysadmin, Cura will provide:

1) A personalized Terminal emulator for direct interaction with said server.

2) The Syslog reader that allows for reading logs directly from the server.

3) A system monitor that visually graphs CPU and RAM usage percentages.

4) Access to the port-scanning utility “Nmap”.

5) Server Stats are general server information like its Vitals (hostname, last boot time, logged in users, etc..), Mounted Filesystems, Memory Information and Processes Status. The Processes Status section has a "Kill Processes" button inside (brings up a dialog listing of the most costly processes, selecting one of these will cause said process to be killed (given that the user has the needed privileges to do so)).

6) Log-in monitoring with notifications (while you are logged in, you can get notifications about anyone else that also logged in to said server)

5) Added to which, a security feature is implemented that allows you to have Cura's database wiped when you send the compromised phone a secret pattern of your choosing. (ex: send an SMS message containing "phone has been stolen!" to your Android phone to wipe Cura's database and receive the location of the compromised phone as an SMS to your emergency phone number or as an e-mail to your emergency e-mail address).

Cura is multi-lingual and for now, it supports English, German and Greek.
